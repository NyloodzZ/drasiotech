Drasiotech : Innovation Web & SEO en BTS SIO

Drasiotech n'est pas qu'un simple projet étudiant. C'est une démonstration technique d'ingénierie web moderne déployée sur une infrastructure Serverless.

Une Architecture SLAM Nouvelle Génération

Contrairement aux sites statiques traditionnels, Drasiotech.dev utilise les derniers standards du web pour garantir une performance maximale (Core Web Vitals).

  • Optimisation SEO : Utilisation de balises canoniques, JSON-LD et sémantique HTML5 stricte.
  • Infrastructure : Hébergement haute disponibilité sur le réseau Edge de Vercel.
  • Sécurité : Domaine en .dev avec HSTS préchargé et certificats SSL/TLS automatiques.

Expertise Technique BTS SIO option SLAM

Ce projet s'inscrit dans le cadre du BTS Services Informatiques aux Organisations, option Solutions Logicielles et Applications Métiers.

Il démontre la maîtrise des compétences clés :

Le développement d'applications web, la gestion de versions avec Git, le déploiement continu (CI/CD) et l'analyse de trafic via la Google Search Console.
